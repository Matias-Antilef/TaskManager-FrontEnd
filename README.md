
# Task Maneger - front end


## Instalación

```bash
  npm install
```
    
## Despliegue

```bash
  npm run dev
```


## Acerca de

    Tecnologías usadas:

    - React + Vite
    - TypeScript
    - Tailwind CSS
    - Axios
    - Context API


## Estructura

    hooks/
        useMongoDB - CRUD para mongoDB
    
    provider/
        TaskProvider - CRUD context API
